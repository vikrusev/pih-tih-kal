# pih-tih-karl
Push the pedal to the metal.

The idea of the project is to have fun with friends and cars.

## Технологии:
- Express.js
- MongoDB 
- ThreeJS
- SCSS

## Функционалности:
- Възможност за игра с други играчи в реално време
- Състезания, upgrades, gameplay
- 3D коли
- Потребителски настройки
